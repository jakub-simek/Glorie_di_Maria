# Italienischer Editionstext

Hier entsteht der italienische Leittext der digitalen Edition.

Vorlaufige Grundlage: IntraText-Fassung von *Glorie di Maria*, nach dem Research-Bericht textgleich mit der Standardausgabe der *Opere ascetiche* VI-VII.

Jeder Abschnitt soll stabile IDs erhalten, damit deutsche Fassung, Kommentare und Meditationen darauf verweisen konnen.
